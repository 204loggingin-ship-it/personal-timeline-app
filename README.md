# 个人应用：日历与时间轴版

这是一个可以安装到 iPhone 主屏幕、断网后继续使用的个人时间轴应用。程序数据默认保存在当前手机中，不会随网站发布到公网。

## 安装到 iPhone

1. 使用 Safari 打开正式的 HTTPS 地址。
2. 等页面完整打开一次。
3. 点击 Safari 底部的“分享”按钮。
4. 选择“添加到主屏幕”，再点击“添加”。
5. 从桌面图标打开一次，然后开启飞行模式验证离线使用。

删除桌面应用、清除 Safari 网站数据或更换手机可能使本地记录消失。请定期从“时间轴”页的导出按钮导出 JSON 完整备份；CSV 主要用于查看表格，不代替完整备份。

当前版本包含：

- 每日、实时、时间轴三个页面
- 实时文字记录与自动时间
- 本地保存
- 月历与日期切换
- 所选日期的时间线向左滑进入同一天备注，向右滑返回
- 时间线与备注切换时保持当前所选日期
- 底部导航固定在屏幕底部
- 有记录日期的小圆点提示
- 每日睡眠、起床、步数与经期填写
- 固定每日记录可修改且不会重复创建
- 时间轴蓝色与橙色标签
- 经期日期红色标记
- 时间轴手动创建记录
- 普通记录编辑与二次确认删除
- 每日固定记录从每日页面统一修改
- 每个日期独立的文字备注
- 备注自动保存、回填与二次确认清空
- 日历备注标记
- JSON 完整备份导出
- CSV 表格导出
- JSON 备份检查与非覆盖式合并导入
- 当日图片备注、缩略图与删除
- 图片自动缩小并保存在专用本地数据库
- 图片随第 2 版 JSON 完整备份导出与恢复
- 首次成功加载后的离线访问

时间直接来自设备系统时钟，不需要网络。

## 在电脑上运行

在本目录启动任意静态网页服务器，然后访问其本地地址。Service Worker 需要通过 `localhost` 或 HTTPS 运行，直接双击 `index.html` 只能查看页面，不能验证离线缓存。

## 第一阶段测试标准

1. 页面能正确显示日期、时间和星期。
2. 时间每秒更新。
3. 页面适合手机竖屏显示。
4. 在线打开一次后，断网并重新打开仍可使用。
